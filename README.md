# todo-agent
Helpful AI Agent that listens to your goal, produces a to-do list, and then executes that to-do list.

## Install dependencies

Run 

```
pip install -r dependencies.txt
```

## API Key

You need an OpenAI API key to run this code.

Create a file called ".env" and write your OpenAI API key in there:

```
OPENAI_API_KEY="sk-proj-<your key here>"
```
